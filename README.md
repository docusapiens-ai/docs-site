## Documentation Structure

This documentation follows the **Diátaxis framework**, which organizes documentation into four quadrants:

- **Tutorials** — Step-by-step guides for learning
- **How-to Guides** — Instructions for accomplishing specific tasks
- **Reference** — Technical specifications and facts
- **Explanation** — Understanding-oriented content, concepts, and "why"

This structure helps you find exactly what you need, whether you're learning or looking up specific information.

## DocuSapiens Platform

DocuSapiens is a **fully managed SaaS platform** that:

✅ Automatically deploys Docusaurus sites from your GitHub repositories  
✅ Hosts your documentation at `<subdomain>.docusapiens.ai`  
✅ Integrates an AI-powered chat assistant trained on your docs  
✅ Syncs updates in real-time with every GitHub commit  
✅ Requires zero configuration — just connect your repo and go live  

Currently in **Alpha** — free access to all features. [Join now](https://app.docusapiens.ai).

## Docusaurus Framework

Docusaurus is an **open-source documentation framework** by Meta that:

✅ Converts Markdown files into a professional documentation site  
✅ Provides built-in search, versioning, i18n, and dark mode  
✅ Is powered by React and Algolia for performance  
✅ Has a clean, modern default theme (customizable)  
✅ Powers documentation for React, Babel, Kubernetes, and 1000+ projects  

Learn more at [docusaurus.io](https://docusaurus.io).
