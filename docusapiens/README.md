## Key Concepts

**Docusaurus** â€” The open-source documentation framework (by Meta) that powers every site you create with DocuSapiens. It converts Markdown files into a professional, searchable website.

**GitHub Sync** â€” DocuSapiens watches your GitHub repository. Every time you commit and push, your site automatically rebuilds with the latest content.

**AI Chat** â€” A retrieval-augmented generation (RAG) system that reads your documentation and answers user questions. It never invents information â€” it only uses what's in your docs.
